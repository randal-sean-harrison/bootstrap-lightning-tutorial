Introducing Bootstrap 3

Download Bootstrap 3.3.7

An HTML + CSS + JS Front-end Framework

We will be using Bootstrap 3.3.7 for this website. While Bootstrap 4 offers exciting new features, it's still in beta. More importantly, it does not support older browsers. For these reasons, it's not recommended for sites which must be accessible to all audiences, on all devices.

Create an HTML Page

Begin with the shell of an HTML5 page, with includes of both Bootstrap 3.3.7 and the latest version of jQuery. I've also inlcuded a link to the Font Awesome Web icons library. Make sure to save your page, if necessary.

<!DOCTYPE html>
<html lang="en">
   <head>

      <!-- Metadata for the Webpage -->
      <meta charset="utf-8">
      <meta http-equiv="X-UA-Compatible" content="IE=edge">
      <meta name="viewport" content="width=device-width, initial-scale=1">
      <title>Title</title>

      <!-- Bootstrap CSS files-->
      <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap.min.css">

      <!-- Font Awesome Web Icons -->
      <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/font-awesome/4.4.0/css/font-awesome.min.css">

   </head>
   <body>


      <!-- [ This is where your body content will go... ] -->


      <!-- jQuery Javascript Library (required by Bootstrap) -->
      <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>
      <!-- Bootstrap Library -->
      <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/js/bootstrap.min.js"></script>

   </body>
</html>

Add a Container (wrapper)

The 12-column Bootstrap grid is built by stacking limitless rows inside of a single wrapping container. (You may nest containers, though typically you'll add a single container, with multiple rows inside of it.)

If whitespace is not your thing, use a full-width layout by changing the class of your wrapping container from class="container" to class="container-fluid".

<div class="container">
   <!-- Your rows go here -->
</div>

3

Add a Jumbotron

Let's add a Jumbotron to the top of the page. To remove its rounded corners and force the jumbotron to take up the full width of the screen, add it before the main .container div.

<div class="jumbotron">

  <h1>Hello, world!</h1>

  <p>Lorem ipsum dolor sit amet, consectetur adipisicing.</p>
  <p>
     <a class="btn btn-primary btn-lg" href="#" role="button">Learn more</a>
  </p>

</div>

4

Add Rows

Add a row with 4 columns (for a large monitor), 2 columns (for a medium device), and 1 column (for a small device) inside the container div.

<div class="row">

   <div class="col-sm-12 col-md-6 col-lg-3">
      <!-- Your Add a thumbnail -->
      Lorem ipsum dolor.
   </div>

   <div class="col-sm-12 col-md-6 col-lg-3">
      <!-- Your Add a thumbnail -->
      Lorem ipsum dolor.
   </div>

   <div class="col-sm-12 col-md-6 col-lg-3">
      <!-- Your middle Column Content Goes Here -->
      Lorem ipsum dolor.
   </div>

   <div class="col-sm-12 col-md-6 col-lg-3">
      <!-- Your right Column Content Goes Here -->
      Lorem ipsum dolor.
   </div>

</div>

5

Don't Forget The Navigation!

Add the navigation element to the page. To shift the navbar darker, add the .navbar-inverse class.

<nav class="navbar">
   <div class="container">

     <div class="navbar-header">
       <a class="navbar-brand" href="#">MY COOL WEBSITE</a>
     </div> <!-- .navbar-header -->

     <div id="navbar">
       <ul class="nav navbar-nav pull-right">
         <li class="active"><a href="index.html">About</a></li>
         <li><a href="teaching.html">Teaching</a></li>
         <li><a href="research.html">Research</a></li>
         <li><a href="contact.html">Contact</a></li>
       </ul>
     </div>

   </div> <!-- .container -->
</nav> <!-- .nav -->
